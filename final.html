정보 보호 컨설팅 기반 네트워크 웹 보안 사업

작업기간 : 2025.11.03 ~ 2025.12.12
작업인원 : 6명 (PL 담당)
사용기술 : VMware, Linux, GNS3, ELK Stack

PL을 맡아 본사/지사로 나누어져있는 개별 인원의 통일성을 맞추기 위해 노력하였고,
웹 서비스를 제공하는 가상의 기업 환경을 설정하고, 보안 침해 시나리오와 인프라 구성도를 기반으로 하여
주요정보통신기반시설 취약점 분석·평가 가이드 를 기준으로
UNIX, 웹 서비스 전반에 대한 취약점 진단 및 보안 설정을 수행한 프로젝트입니다.
또한 PHP 기반 웹 서비스에 대한 취약점 진단을 수행했습니다.
본 프로젝트에서 CentOS 서버 설치/점검, Rocky Linux 설치/보안/점검, 정책서 작성,
ELK Stack 기반 LOG 모니터링 시스템 구축을 담당했습니다.

  • CentOS/Rocky Linux 취약점 진단 및 보안 조치
  • ELK Stack 기반 실시간 로그 모니터링 시스템 구축
  • 보안 정책서 및 가이드라인 작성
